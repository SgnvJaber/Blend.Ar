Description:
The following contains a mini website page,upon clicking
	on "הצג את רשימת השירים שלי" an ajax call to "music_list.php"
	which contains the music list data,will generate a table containing
	a list of music,and on the right side a button "play",to play the music.
	clicking on play will call the video and play it using the Youtube API.
	also,clicking on "הסתר" which originally was "הצג את רשימת השירים שלי"
	will refresh the page.
	The web-page also automatically makes an ajax call 
	to "get_current_date.php" which will display the Date on the right	corner of the site.
